# 二 开发工具之 Sublime
1. 常用快捷键：
- 查找（ctrl+p）
- 命令面板（ctrl+shift+p）
- 多行选择（ctrl+d,ctrl+shift+l）

2. 常用插件（前端）
- Emmet
- DocBlockr
- SideBarEnhancements
- Terminal

3. Snippets
建立代码片段用来快速使用复用代码。（工具>插件开发>新建代码片段）
![开发工具](图2.PNG)

# 三 调试工具
- Elements：用来调试 HTML，CSS。用来增删改的操作。
- Network：网络请求。
- Sources：网站的所有资源。
- Console：与 Sources 结合来调试 JS。
- Resources：可以看到网站的本地数据资源。
- Timeline，Profiles，Audits 用来调试页面性能。
