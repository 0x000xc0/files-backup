# 基础格式
使用 `<hgroup>` 标签对网页或区段（section）的标题进行组合，逻辑上为一体：
```
<hgroup>
 <h1>Welcome to my WWF</h1>
 <h2>For a living planet</h2>
</hgroup>
```

上标，下标
```
a<sup>2</sup>+b<sub>0</sub>
```

加粗
```
<b></b>
```
 
斜体
```
<i></i>
```

缩小
```
<small></small>
```

删除
```
<del></del>
```

增加的
```
<ins></ins>
```

过时的不提倡使用
```
<s></s>
```

高亮
```
<mark></mark>
```

定义
```
<dfn>define</dfn>
```

代码片段
```
<code>code代码</code>

<samp>samp例子代码</samp>
```

用户输入
```
<kbd>kbd用户输入</kbd>
```

变量
```
<var>变量</var>
```

引用
```
<cite>cite引用</cite>
```

地址
```
<address>Rm401 XK West</address>
```

引用
```
<q>：引用的是小段文字；

<blockquote>：引用的是大段的内容块。
```

预格式化不做处理
```
<pre></pre>
```

水平线
```
<hr>
```

缩写，鼠标停留有注释
```
<abbr="中华人民共和国">PRC</abbr>
```
